# Alura-Store

## Descripción General
Este proyecto tiene como objetivo realizar un análisis de los datos de venta de cuatro tiendas de Alura Store, con el propósito de identificar cuál es la tienda con menor rendimiento obteniendo información útil para la toma de decisiones.

El conjunto de datos incluye información detallada de los productos vendidos, categorías, precios, costo de envío, calilficación de los clientes, etc. A partir de estos se desarrolló un análisis integral utilizando herramientas de data science y visualización.

## Objetivos del Proyecto
* Evaluar el rendimiento comercial de cada tienda.
* Analizar los productos y categorías más vendidos.
* Calcular los ingresos totales y el costo de envío.
* Determinar las calificaciones promedio de los clientes.
* Identificar los productos más y menos vendidos.
* Realizar las gráficas necesarias para representar los objetivos medidos.

## Tecnologías Utilizadas
* Python: Lenguaje principal utilizado para la manipulación y an{alisis de los datos.
* Pandas: Librería empleada para la gestión de estructuras de datos.
* Matpltlib: Herramienta utilizada para la creación de gráficos.
* Github: Plataforma utilizada para el almacenamiento del proyecto.

## Características y Funcionalidades
* Se importaron los datos desde fuentes externas.
* Se identificaron patrones de venta, categorías destacadas y nivel de satisfacción de los clientes.
* Se utilizaron gráficos de barra para representar los principales indicadores.
* Se compararon los resultados de las cuatro tiendas para determinar cuál presenta el menor rendimiento.

## Conclusión
Nuestro análisis obtuvo una visión completa del desempeño de cada tienda, tanto en términos financieros como de satisfacción de los clientes, llegando a la conclusión de que la tienda 4 es la menos eficiente y se recomienda cerrar.
